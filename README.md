## 📋 О проекте

**ПК мастер** - это полноценный интернет-магазин электронной техники, разработанный в рамках курсового проекта. Сайт предоставляет пользователям возможность просматривать каталог товаров, добавлять их в корзину и оформлять заказы. Для менеджеров и администраторов предусмотрена панель управления для обработки заказов и управления ассортиментом.

### 🎯 Целевая аудитория
- Покупатели электронной техники
- Менеджеры интернет-магазина
- Администраторы системы

## 👥 Роли пользователей

| Роль | Права |
|------|-------|
| **Гость** | Просмотр каталога и отдельных товаров |
| **Покупатель** | Всё, что может гость + добавление в корзину, оформление заказов, история заказов |
| **Менеджер** | Управление заказами, добавление/редактирование товаров |
| **Администратор** | Полный доступ, управление пользователями и ролями |

## 📖 User Stories

### Гость
- [ ] Я хочу просматривать каталог товаров с фильтрацией по категориям
- [ ] Я хочу видеть подробную информацию о товаре (характеристики, цена, описание)
- [ ] Я хочу зарегистрироваться на сайте

### Покупатель
- [ ] Я хочу добавлять товары в корзину
- [ ] Я хочу оформить заказ с указанием адреса доставки
- [ ] Я хочу просматривать историю своих заказов

### Менеджер
- [ ] Я хочу добавлять новые товары в каталог
- [ ] Я хочу изменять статусы заказов

### Администратор
- [ ] Я хочу управлять пользователями (блокировать, изменять роли)

## 🗄️ Модели данных

### Основные сущности:

```python
# Пользователь (User)
- id: int (первичный ключ)
- username: str (уникальный)
- email: str (уникальный)
- hashed_password: str
- role: str (customer/manager/admin)
- is_active: bool
- created_at: datetime

# Категория (Category)
- id: int
- name: str
- description: text

# Товар (Product)
- id: int
- name: str
- description: text
- price: float
- stock_quantity: int
- category_id: int (внешний ключ)
- image_url: str

# Корзина (Cart)
- id: int
- user_id: int (внешний ключ)
- created_at: datetime

# Элемент корзины (CartItem)
- id: int
- cart_id: int (внешний ключ)
- product_id: int (внешний ключ)
- quantity: int

# Заказ (Order)
- id: int
- user_id: int (внешний ключ)
- order_date: datetime
- total_amount: float
- status: str (pending/processing/shipped/delivered/cancelled)
- shipping_address: text
- phone_number: str

# Элемент заказа (OrderItem)
- id: int
- order_id: int (внешний ключ)
- product_id: int (внешний ключ)
- quantity: int
- price: float
